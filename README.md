# 🎓 Student Portal

A Flask-based Student Portal developed to help colleges manage student information digitally. The system allows administrators to manage student records, upload student data through Excel, take attendance, manage marks, and generate attendance reports. Students can securely log in using their USN to view their personal information, attendance, and marks.

---

## 🚀 Features

- 🔐 Admin Login
- 👨‍🎓 Student Management
  - Add Student
  - Edit Student
  - Delete Student
  - Search Student
  - Student Profile
  - Photo Upload
- 📤 Upload Students from Excel
- 📅 Attendance Management
- 📄 Attendance Report (PDF Download)
- 📝 Marks Management
- 🔍 Search Students
- 👤 Student Login (USN Based)

## 🎯 Project Objective

The objective of this project is to provide a simple, secure, and user-friendly Student Management System for educational institutions. It helps administrators manage student records, attendance, and academic information efficiently while allowing students to access their own information securely.
